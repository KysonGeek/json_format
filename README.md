# JSON工具

一个简单而功能强大的JSON工具页面，提供JSON校验、格式化、压缩、转义和去除转义功能。

## 功能特点

- **JSON校验与格式化**：检查JSON格式是否正确，并进行美化格式化
- **JSON压缩**：移除所有空格，生成紧凑的JSON字符串
- **字符串转义**：将JSON字符串中的特殊字符进行转义处理
- **去除转义**：移除JSON字符串中的转义字符
- **语法高亮**：使用不同颜色显示JSON的不同部分（键、值、数字、布尔值等）
- **行号显示**：在编辑器左侧显示行号，方便定位
- **JSON折叠**：支持折叠/展开JSON的对象和数组，方便查看大型JSON

## 使用方法

1. 在文本框中输入或粘贴JSON字符串
2. 点击相应的按钮执行操作：
   - **格式化**：检查JSON格式并美化显示（支持语法高亮和折叠）
   - **压缩**：生成无空格的紧凑JSON
   - **转义**：对字符串进行转义处理
   - **去除转义**：移除转义字符

## 快捷键

- **Ctrl+Enter** 或 **Cmd+Enter**：校验并格式化JSON
- **Ctrl+Shift+C** 或 **Cmd+Shift+C**：压缩JSON

## 使用提示

- 校验成功后，双击高亮显示区域可返回到编辑模式
- 点击对象或数组前的箭头图标可折叠/展开该部分
- 状态信息会在操作完成后显示，并在3秒后自动消失

## 文件结构

- `index.html`：主HTML文件
- `styles.css`：样式文件
- `script.js`：JavaScript功能实现

## 本地运行

直接在浏览器中打开`index.html`文件即可使用该工具。
